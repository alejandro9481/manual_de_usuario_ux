Cómo Subir Manga

Introducción

En este módulo encontraras toda la información necesaria para subir un manga a MangaDex, ya sea como parte de un grupo de traducción o de manera independiente. Asegúrate de seguir todas las reglas y requisitos para garantizar que tu contenido cumpla con las normas de la plataforma y sea aprobado sin problemas.

Requisitos previos para Subir Manga

Antes de subir un manga, asegúrate de cumplir con estos requisitos:

    • Ser miembro de un grupo de traducción en MangaDex o crear uno propio.
    • Cumplir con las normas sobre el tipo de contenido permitido para evitar infracciones.
    • Verificar que el capítulo no viole derechos de autor y que respete la política de MangaDex.

Pasos para Subir un Capítulo de Manga

  1. Inicia sesión: Inicia sesión en tu cuenta de MangaDex y dirígete a la serie en la que deseas subir el capítulo.
  2. Haz clic en "Upload Chapter": Haz clic en el botón "Upload Chapter" para iniciar el proceso de subida.
  3. Selecciona tu grupo de traducción: Selecciona tu grupo de traducción o marca la opción "No afiliado a ningún grupo" si subes el capítulo de forma independiente.
  4. Rellena los datos del capítulo:
    • Idioma: Selecciona el idioma del capítulo.
    • Número de Volumen y Capítulo: Asegúrate de introducirlos correctamente.
    • Título del Capítulo: Opcionalmente, puedes incluir un título descriptivo.
    • One-shot: Marca esta opción solo si es un capítulo único.
  5. Sube las imágenes: Arrastra los archivos o selecciónalos desde tu dispositivo. MangaDex admite formatos de imagen como JPEG y PNG.
  6. Revisa y publica: Revisa todos los datos y haz clic en "Upload" para publicar el capítulo.

Detalles y Reglas de Subida

Para asegurar la calidad y cumplimiento, considera las siguientes reglas:

    • No subas contenido oficial sin permiso ni material que no sea traducción.
    • Los capítulos deben mantener la estructura original (sin combinaciones o divisiones arbitrarias).
    • Las páginas adicionales (como créditos o fan arts) deben colocarse al inicio o final del capítulo.
    • No uses el nombre del capítulo para añadir etiquetas innecesarias como "HQ" o "v2".

Al publicar un capítulo, este puede necesitar aprobación del equipo de MangaDex. Este proceso tarda entre 24 y 48 horas, especialmente si es la primera vez que subes contenido.

Restricciones de Contenido

Para mantener la calidad y el cumplimiento con las políticas de MangaDex, ten en cuenta las siguientes restricciones de contenido:

    • Solo se permiten títulos de estilo cómic japonés, coreano, chino o de origen asiático similar. Excepciones pueden hacerse para cómics autopublicados de estilo similar con previa autorización del personal de MangaDex.
    • Solo se permiten subir scanlations (cómics traducidos y editados) y cómics originales autopublicados. No se permiten compilaciones de arte sin texto para traducir.
    • Los escaneos de lanzamientos oficiales (ya sea físico o digital) no están permitidos, salvo excepciones como la expiración de licencias o permiso del editor.
    • No se permiten subidas que contengan marcas de agua de agregadores ni contenido generado automáticamente sin suficiente edición manual.
    • Las subidas deben respetar la política de restricciones de grupos: si un grupo tiene restricciones de subida, solo sus miembros pueden subir capítulos.

Detalles Técnicos

Para asegurar la calidad de las imágenes y cumplir con las especificaciones de MangaDex, ten en cuenta los siguientes detalles técnicos:

    • Formato de archivo: Solo se permiten imágenes en formato JPG, JPEG, PNG o GIF.
    • Orientación: Las imágenes deben tener orientación vertical.
    • Resolución: Utiliza la mayor resolución posible (mínimo 300px x 450px).
    • Tamaño de imagen: Cada imagen no debe superar los 20 MB de tamaño ni exceder 10000 px x 10000 px en resolución.
    • Límite por capítulo: Un capítulo no debe exceder 500 imágenes o 200 MB en total.

Subir como Parte de un Grupo vs. Independiente

En MangaDex, puedes subir capítulos como parte de un grupo de traducción o de manera independiente. A continuación, se explican las diferencias:

Subir como Parte de un Grupo

Si eres miembro de un grupo de traducción, puedes seleccionar el grupo al subir un capítulo. Esto permite que el grupo reciba reconocimiento por su trabajo, y es ideal para equipos que colaboran en la traducción de múltiples mangas.

Subir de Manera Independiente

Si no perteneces a ningún grupo, puedes subir capítulos de manera independiente seleccionando la opción "No afiliado a ningún grupo". Esta opción es adecuada para usuarios que desean contribuir con un capítulo sin formar parte de un equipo.
